Projet Bibliobus

Le projet Bibliobus vise à faciliter l'accès à la culture en mettant en place un bus itinérant qui permet aux abonnés de réserver et d'emprunter des livres via une application numérique le bibliothécaire gère les prêts et retours ainsi que l'ajout de nouveaux ouvrages dans la base de données le projet implique la conception d'une base de données relationnelle une application web et une application desktop avec une API pour la gestion des livres et des utilisateurs
